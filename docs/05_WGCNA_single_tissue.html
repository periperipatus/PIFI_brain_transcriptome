1 Gonads (GON)

Like the DESeq2 document, the Gonads section will contain the code used throughout the the analysis to achieve the full set of results. First, each dataset is filtered conservatively as per the DESeq2 workflow, and then any batch effects identified in the DEseq workflow are removed here. Then we create a midweight bicorrelation weighted signed network using the recommended options. The Topological Overlap Matrix is also signed.
