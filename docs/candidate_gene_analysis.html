For manuscript: Neurogenomic landscape of male cooperative behavior in a wild bird

Peri Bolton

2022-01-13

This document contains the differential expression results from DESeq2 for our candidate genes. These candidates are: AR, SRD5A2, LOC113993669, ESR1, ESR2, PGR, GNRH1, PRL, PRLR, VIP, VIPR1, VIPR2, LOC113983511, OXTR, LOC113983498, AVPR1A, AVPR1B, LOC113982601. LOC113983511 is OXT (Mesotocin), LOC113993669 is CYP19A1 (Aromatase), and LOC113983498 is vasotocin-neurophysin (AVP or Arginine-Vasotocin). Originally I was correcting for multiple comparisons using the raw p-values for the number of genes expressed in a tissue (using Benjamini-Hochberg). However, these were more conservative than the corrected p-values from DESeq2. Therefore, i only present the corrected p-values from DESeq2 based on the number of genes expressed in that tissue.

Status specific gene expression in the Gonads

If not already identified as a possible interaction from multiple incidences in the single trait comparison, I also used DESeq2 to extract possible interactions.

Then, I am going to use AIC on simple linear models to decide amongst models to decide whether interaction effects or single term models better explain the data.

Note that the DESeq2 model for Status only is actually GeneExp ~ Batch + Status, and includes 4 additional samples. However, the linear models here don’t differ in biological interpretation.

VIPR_GON_lm_Status<- lm(VIPR2 ~ Year + Status, data=VIPR2)
VIPR_GON_lm_meanT<- lm(VIPR2 ~ Year + mean_T, data=VIPR2)
VIPR_GON_lm_int<- lm(VIPR2 ~ Year + Status*mean_T, data=VIPR2)
#against the NULL model
VIPR_GON_lm_null<- lm(VIPR2 ~ Year, data=VIPR2)
